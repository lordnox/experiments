Carousel based on Bustle.com

Converted from ember.js to a partial angular.js version. Angular is only used to do the set the variables in the article-block class [`on-deck`, `in-the-hole`, `last-up`, `at-bat`]. Because this is not using an angular directive we need to tell angular the size and position of the first element.

`ng-init=""` will do this. Additionaly every slide needs a reference to the slide number so that angular can access it.

Additionaly the is a `slider.direction` property that will define the direction the slides should fade. It will be set by the next/prev-buttons.

For debugging you can turn of the hidden-property on the carousel:
